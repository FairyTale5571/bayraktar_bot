Правила использования админки

1. Не использовать ГМ в игровых ситуациях

2. Не телепертироваться к игроку просто так

3. Не телепертировать игрока к себе просто так

4. Не спавнить вещи для собственной выгоды

5. Не спавнить вещи для других игроков

6. Не убивать игроков в качестве наказаний! Только бан!

7. Не использовать админку для выгоды себе или другим игрокам

8. Если что то не понятно, обращаться к админам лично
